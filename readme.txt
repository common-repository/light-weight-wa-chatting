== Description ==
Customize the WordPress theme with powerful, professional, and intuitive WhatsApp chatting capability without using any library or framework.

== Frequently Asked Questions ==
*How To Install?*
1. Upload the plugin folder to the "/wp-content/plugins/" directory, or install the plugin directly through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Check the WhatsApp Tab on your WordPress Admin

*How To Use?*
1: Install & Activate The Plug-in.
2: Go To the Plugin Settings Page In The Administrator Menu Called WhatsApp.
3: Enter A Valid Phone Number In The Format Of "962770502854".
4: Choose a Theme Select List Item.
5: Choose Position Select List Item.
6: Check "With Entrance Animation" If You Would Like The Chatting Icon To Enter The Website Pages With Entrance Animation.
7: Click The "Save" Button.
8: Done!, Go To the Website Front-end To See The Result Of Your Settings.
